# update what will be committed
```
git add <file>...
```
Usually use the following command to update current folder:
```
git add .
```

# discard changes in the working directory
```
git checkout -- <file>...
```
# file change history
```
gitk <file>
```
or:
```
gitk --follow <file>
```
For example:
```
gitk cwm-ng/src/app/app.module.ts
```
